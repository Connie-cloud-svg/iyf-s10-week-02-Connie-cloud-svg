## Project Description
A fully responsive personal portfolio website built with HTML5 and CSS3.

## Pages
- **Home** - Hero section with profile photo and skills grid
- **Projects** - 6 project cards with categories
- **Contact** - Contact form with sidebar

## Features
- Mobile-first responsive design
- CSS-only hamburger menu
- Dark mode support
- Hover effects and smooth transitions
- Emerald Dev colour theme

## Technologies Used
- HTML5 (Semantic)
- CSS3 (Flexbox, Grid, Media Queries)
